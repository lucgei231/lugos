LucasOS

LucasOS is a web-based operating system that is designed to be simple and easy to use.

It is built using HTML, CSS, and JavaScript.

LucasOS is open-source and free to use.

It is licensed under the MIT License.

LucasOS is created by Lucas Santos.

LucasOS is hosted on GitHub Pages.

LucasOS is powered by Firebase.
